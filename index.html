⚠ Bases de datos requeridas — no incluidas en el repositorio
El modelo depende de archivos de datos externos (SAM, elasticidades, datos sectoriales) construidos por el equipo técnico. Estos archivos no están en este repositorio por razones de confidencialidad y tamaño. Son entregados directamente al cliente junto con este repositorio.

Una vez recibidos, ubíquelos en la carpeta data/ en la raíz del proyecto. Consulte el manual de usuario para más detalles.
